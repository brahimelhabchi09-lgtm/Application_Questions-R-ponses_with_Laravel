# QA-Frontend (React + Vite)

This is the modern, decoupled frontend for the Questions & Reponses platform.

## 🚀 Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## 🎨 Theme & UI

- **Framework**: React 18
- **Styling**: Tailwind CSS (Premium Glassmorphism theme)
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Fonts**: Outfit (Google Fonts)

## 📡 API Connection

The frontend is configured to proxy requests to `http://localhost:8000/api` via the `vite.config.js` settings.
Make sure your Laravel server is running on port 8000.
